# ESF-8 University Platform

## Overview
The ESF-8 University Platform is a modernized public health and medical response resource hub designed to support emergency preparedness, training, and real-time situational awareness.

This platform builds upon the ESF-8 (Public Health & Medical Services) framework to provide accessible tools and resources for students, professionals, and response organizations.

## Key Features
- 🧠 **Training Resources**  
  Educational materials focused on public health emergency response and preparedness.

- 📊 **VMOC Integration**  
  Weekly updates and situational awareness reports to support real-time decision-making.

- 🌍 **Disaster & Emergency Resources**  
  Centralized information for responding to public health emergencies and disasters.

- 🤝 **Collaborative Platform**  
  Developed across academic and professional institutions to support coordinated response efforts.

## Purpose
This project aims to modernize and expand the ESF-8 University website by improving accessibility, usability, and integration of real-time public health data.
